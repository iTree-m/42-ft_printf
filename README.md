# ft_printf
- Recoding printf.

## Features
- Manage the following conversions: ```cspdiuxX%```.
-  Manage any combination of the following flags: ```-0.*``` and minimum field
width with all conversions.

## Usage

```c
/* go to ft_printf folder */

make
gcc main.c libftprintf.a
./a.out
```
